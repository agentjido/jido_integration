# `Jido.Integration.V2.GeneratedPlugin`

Macro for generating a connector-level `Jido.Plugin` bundle from authored
manifest truth.

The generated plugin is a derivative consumer surface. It only includes
authored entries published as common consumer surfaces and does not widen
connector-local inventory into shared Jido surface area by itself.
